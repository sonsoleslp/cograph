# Calculate Bezier Curve Points

Calculate points along a quadratic Bezier curve.

## Usage

``` r
bezier_points(x0, y0, x1, y1, x2, y2, n = 50)
```

## Arguments

- x0, y0:

  Start point.

- x1, y1:

  Control point.

- x2, y2:

  End point.

- n:

  Number of points to generate.

## Value

Data frame with x, y coordinates.
