# Generate Cross/Plus Vertices

Generate Cross/Plus Vertices

## Usage

``` r
cross_vertices(x, y, r, thickness = 0.3)
```

## Arguments

- x:

  Center x coordinate.

- y:

  Center y coordinate.

- r:

  Half-size.

- thickness:

  Arm thickness as ratio of r.

## Value

List with x, y vectors of vertices.
