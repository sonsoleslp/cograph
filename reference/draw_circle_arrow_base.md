# Draw Circle Arrow (Dot)

Draws a circular dot at the arrow position (alternative to triangular
arrow).

## Usage

``` r
draw_circle_arrow_base(x, y, size, col = "black", border = NULL)
```

## Arguments

- x:

  Position x coordinate.

- y:

  Position y coordinate.

- size:

  Dot radius.

- col:

  Fill color.

- border:

  Border color.
