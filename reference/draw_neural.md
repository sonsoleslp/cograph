# Draw Neural Node

Circle with small connection circles around the perimeter (neuron-like).

## Usage

``` r
draw_neural(
  x,
  y,
  size,
  fill,
  border_color,
  border_width,
  alpha = 1,
  n_connections = 6,
  ...
)
```

## Arguments

- n_connections:

  Number of connection points around perimeter.
