# Draw Self-Loop

Draw Self-Loop

## Usage

``` r
draw_self_loop(x, y, node_size, color, width, lty, rotation = pi/2)
```

## Arguments

- x, y:

  Node center coordinates.

- node_size:

  Node radius.

- color:

  Loop color.

- width:

  Loop line width.

- lty:

  Loop line type.

- rotation:

  Angle in radians for loop direction (default: pi/2 = top).
