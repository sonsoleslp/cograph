# Generate Ellipse Vertices

Generate Ellipse Vertices

## Usage

``` r
ellipse_vertices(x, y, rx, ry, n = 50)
```

## Arguments

- x:

  Center x coordinate.

- y:

  Center y coordinate.

- rx:

  Horizontal radius.

- ry:

  Vertical radius.

- n:

  Number of vertices.

## Value

List with x, y vectors of vertices.
