# Format P-value

Formats a p-value with specified decimal places and prefix.

## Usage

``` r
format_pvalue(p, digits = 3, prefix = "p=")
```

## Arguments

- p:

  Numeric p-value.

- digits:

  Number of decimal places.

- prefix:

  Prefix string (e.g., "p=").

## Value

Formatted p-value string.
