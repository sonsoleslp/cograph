# List Available Layouts

List Available Layouts

## Usage

``` r
list_layouts()
```

## Value

Character vector of registered layout names.

## Examples

``` r
list_layouts()
#>  [1] "circle"               "oval"                 "ellipse"             
#>  [4] "spring"               "fr"                   "fruchterman-reingold"
#>  [7] "groups"               "grid"                 "random"              
#> [10] "star"                 "bipartite"            "custom"              
#> [13] "gephi_fr"             "gephi"               
```
