# List Available Color Palettes

Returns the names of all registered color palettes.

## Usage

``` r
list_palettes()
```

## Value

Character vector of palette names.

## Examples

``` r
list_palettes()
#> [1] "rainbow"    "colorblind" "pastel"     "viridis"    "blues"     
#> [6] "reds"       "diverging" 
```
