# Offset Point from Center

Calculate a point offset from another point by a given distance.

## Usage

``` r
offset_point(x, y, toward_x, toward_y, offset)
```

## Arguments

- x, y:

  Original point.

- toward_x, toward_y:

  Point to offset toward.

- offset:

  Distance to offset.

## Value

List with x, y coordinates.
