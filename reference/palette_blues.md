# Blues Palette

Generate a blue sequential palette.

## Usage

``` r
palette_blues(n, alpha = 1)
```

## Arguments

- n:

  Number of colors to generate.

- alpha:

  Transparency (0-1).

## Value

Character vector of colors.

## Examples

``` r
palette_blues(5)
#> [1] "#F7FBFF" "#CCDEF1" "#84BBDB" "#3989C1" "#084594"
```
