# Pastel Palette

Generate a soft pastel color palette.

## Usage

``` r
palette_pastel(n, alpha = 1)
```

## Arguments

- n:

  Number of colors to generate.

- alpha:

  Transparency (0-1).

## Value

Character vector of colors.

## Examples

``` r
palette_pastel(5)
#> [1] "#FFB3BA" "#BAFFC9" "#BAE1FF" "#FFFFBA" "#FFDFBA"
```
