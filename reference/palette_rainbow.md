# Rainbow Palette

Generate a rainbow color palette.

## Usage

``` r
palette_rainbow(n, alpha = 1)
```

## Arguments

- n:

  Number of colors to generate.

- alpha:

  Transparency (0-1).

## Value

Character vector of colors.

## Examples

``` r
palette_rainbow(5)
#> [1] "#FF0000FF" "#CCFF00FF" "#00FF66FF" "#0066FFFF" "#CC00FFFF"
```
