# Reds Palette

Generate a red sequential palette.

## Usage

``` r
palette_reds(n, alpha = 1)
```

## Arguments

- n:

  Number of colors to generate.

- alpha:

  Transparency (0-1).

## Value

Character vector of colors.

## Examples

``` r
palette_reds(5)
#> [1] "#FFF5F0" "#FCC4AD" "#FB7D5D" "#E53228" "#99000D"
```
