# Generate Regular Polygon Vertices

Generate Regular Polygon Vertices

## Usage

``` r
regular_polygon_vertices(x, y, r, n, rotation = pi/2)
```

## Arguments

- x:

  Center x coordinate.

- y:

  Center y coordinate.

- r:

  Radius.

- n:

  Number of sides.

- rotation:

  Starting angle in radians (default: first vertex at top).

## Value

List with x, y vectors of vertices.
