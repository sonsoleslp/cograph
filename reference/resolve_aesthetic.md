# Resolve Aesthetic Value

Resolve an aesthetic value that could be a constant, vector, or column
name.

## Usage

``` r
resolve_aesthetic(value, data = NULL, n = NULL, default = NULL)
```

## Arguments

- value:

  Value to resolve.

- data:

  Data frame to look up column names.

- n:

  Expected length.

- default:

  Default value if NULL.

## Value

Resolved vector of values.
