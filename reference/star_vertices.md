# Generate Star Vertices

Generate Star Vertices

## Usage

``` r
star_vertices(x, y, r, n_points = 5, inner_ratio = 0.4)
```

## Arguments

- x:

  Center x coordinate.

- y:

  Center y coordinate.

- r:

  Outer radius.

- n_points:

  Number of star points.

- inner_ratio:

  Ratio of inner to outer radius.

## Value

List with x, y vectors of vertices.
