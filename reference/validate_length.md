# Validate Length Match

Validate Length Match

## Usage

``` r
validate_length(x, expected_length, arg_name = "value", allow_single = TRUE)
```

## Arguments

- x:

  Vector to validate.

- expected_length:

  Expected length.

- arg_name:

  Argument name for error messages.

- allow_single:

  Allow single value (will be recycled).
